Heap Objects

 
The oops directory contains implementation of all heap objects:

instanceOop

    instanceOop -> klassOop[instanceKlass] -> the klassOop [instanceKlassKlass] -> *

objArrayOop

    objArrayOop -> klassOop[objArrayKlass] -> the klassOop [objArrayKlassKlass] -> *

methodOop

    methodOop -> the klassOop[methodKlass] -> *

symbolOop

    symbolOop -> klassOop[symbolKlass] -> the klassOop [typeArrayKlassKlass] -> *

typeArrayOop

    typeArrayOop -> klassOop[typeArrayKlass] -> the klassOop [typeArrayKlassKlass] -> *

constantPoolOop

    constantPoolOop -> klassOop[constantPoolKlass] -> *

vframeArrayOop

     vframeArrayOop -> klassOop[vframeArrayKlass] -> *
 
 * denotes the klassOop[klassKlass] which is the fixpoint for the klass chain.
